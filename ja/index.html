    研究 / 学歴

    研究内容

    卒業論文では再構成可能なハードウェア (FPGA) を用いたバイラテラルフィルタという画像処理フィルタの高速化に取り組みました。医療や自動運転など様々な場面で必要となる画像処理技術ですが、4K や 8K のような高解像度な画像を小規模かつ高速に処理するのは困難です。そこで、並列化やパイプライン化と呼ばれる技術を駆使して、高速に処理できる構造を提案し、それを実際に FPGA 上に実装し、既存手法より小規模で高速に動作することを検証しました。

    さらに、画像処理技術には近年機械学習という技術がよく取り入れられていますが、既存の技術に比べて計算回数が多く、処理に時間がかかってしまうことが課題となっています。そこで、今後は機械学習を用いた画像や点群データの処理に特化して高速化できるハードウェアの仕組みを構築することで社会に役立つ技術の創出を目指します。

    ポートフォリオ

    FADEC
    NNgen、Vivado、Cython、PYNQ を用いた FADEC の実装。「FADEC: FPGA-based Acceleration of Video Depth Estimation by HW/SW Co-design」と題した論文を FPT 2022 にて発表。

    Polylemma
    React、Ethers.js を用いたブロックチェーン上でのPvPコマンドバトルゲーム「Polylemma」のクライアントサイドの実装。「東京 web3 ハッカソン」に出場。

    Replacer
    JavaScript を用いた自動で同時に複数の条件を適用して文字列を変換するためのアプリケーションの実装。デモや実装の内容はこちらに公開中。

    Nested UNet を用いた Building Detection
    PyTorch を用いた Building Detection のための Nested UNet の実装。Fully Connected CRF を用いたり、ノイズ除去を行ったりすることで性能を向上させた。参考論文: “UNet++: A Nested U-Net Architecture for Medical Image Segmentation” by Zongwei Zhou et al. (Springer International Publishing 2018)

    複数画像の PDF 変換
    Python を用いて複数の画像を PDF 変換した上で、結合するプログラムの実装。

    4段パイプライン特権命令付き RISC-V プロセッサ
    System Verilog を用いた 4段パイプライン特権命令付き RISC-V プロセッサ の実装。2レベル適応予測器 を用いた分岐予測や フォワーディング機構 も実装し、ストールを減らして高速化した。CSR 転送命令も簡易的に実装し、例外や割り込み処理 のための特権命令 (mret) も実装した。

    LiDAR セグメンテーションのための ChipNet
    PyTorch を用いた LiDAR 点群から道路のセグメンテーションを行うための ChipNet の実装。KITTI の Road のデータセットを用いて実験を行った。参考論文: “ChipNet: Real-Time LiDAR Processing for Drivable Region Segmentation on an FPGA” by Yecheng Lyu et al. (TCAS1 2019)

    ResNet
    PyTorch を用いた物体識別を行う ResNet の実装。CIFAR-10 のテストデータで 96.01 % の精度を達成した。参考論文: “Deep Residual Learning for Image Recognition” by Kaiming He et al. (CVPR 2016)

    多層パーセプトロン
    Python を用いた手書き文字認識を行う多層パーセプトロン (MLP) のフルスクラッチ実装。MNIST のテストデータで 98.78 % の精度を達成した。

    機械学習手法のフルスクラッチ
    Python を用いた回帰や分類などの様々な機械学習手法のフルスクラッチ実装。

    Fully Pipelined Bilateral Grid
    Vitis HLS、Vivado、PYNQ を用いた FPGA 上での Fully Pipelined Bilateral Grid の実装。「An FPGA-Based Fully Pipelined Bilateral Grid for Real-Time Image Denoising」と題した論文を FPL 2021 にて発表。

    FPGA 上での Gaussian Filter
    Vitis HLS、Vivado、PYNQ を用いた FPGA 上でのガウシアンフィルタの実装。

    画像説明文生成用 AI
    PyTorch を用いた画像説明文生成のための AI。参考論文: “Show and Tell: A Neural Image Caption Generator” by Vinayls et al. (ICML 2015)

    Edge Layout Game
    JavaScript と HTML / CSS を用いた UI を考慮したゲーム開発。こちらに公開中。

    CG 実装
    JavaScript と HTML / CSS を用いたパラメトリック曲線やレイトレーシングや画像処理フィルタなど CG に関する様々な実装。こちらに公開中

    Synner
    JavaScript と HTML / CSS を用いた自動データ生成を支援する Web ページの作成。参考論文: “Is this Real? Generating Synthetic Data that Looks Real” by Mannino et al. (UIST 2019)。こちらに公開中。

    RISC-V CPU
